### 基于LLM大模型的AI机器人
一套基于开源框架、平台的AI语言模型机器人，集成人机对话，信息检索生成，PDF和URL解析对话等功能。

## 工具和平台
Langchain, Streamlit, Oracle Cloud, Groq, Docker
